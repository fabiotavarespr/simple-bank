## Make's Commands
### Start docker-compose
```shell
make docker-compose-up
```

### Stop docker-compose
```shell
make docker-compose-down
```

### Migrate Up
```shell
make migrate-up
```

### Migrate Down
```shell
make migrate-up
```

### Run tests
```shell
make go-test
```

### Run tests with coverage
```shell
make go-test-coverage
```
